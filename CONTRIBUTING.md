# Contributing

Thank you for your interest in contributing to `@hailbytes/mcp-security-scanner`.

**Contributions are not yet open during incubation.**

This package is in early development and the API, architecture, and scope are still being defined internally. We are not accepting external pull requests or issues at this time.

Once the package reaches a stable v0.x release and is published to npm, we will open a contribution guide here. Watch [hailbytes.com](https://hailbytes.com) or the [GitHub repository](https://github.com/HailBytes/mcp-security-scanner) for updates.

— The HailBytes Team
